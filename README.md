# spbu-ipa-course

Этот репозиторий содержит выполненные мной практические задания в рамках курса  **"Алгоритмы обработки изображений"**.

По ходу работы выполнялись в Google Collab, для локального запуска могут понадобится незначительные изменения.

## Список заданий

| № | Тема |
|---|------|
| 1 | IPA basics |
| 2 | Базовые операции анализа изображений|
| 3 | Conditional GAN для генерации изображений собак |
| 4 | Алгоритмы сегментации изображений|
| Отчет | MLPMixer и ConvMixer|
| Доклад | Одноэтапные детекторы|


Подробное описание задания находится в ноутбуке решения.
